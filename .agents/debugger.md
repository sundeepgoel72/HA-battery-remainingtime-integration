# Debugger Agent Role

Own:

- bug reproduction
- regression isolation
- runtime and field validation
- log review and failure characterization
- updates to `.codex/HANDOVER.md` and `.codex/WORKLOG/`

Before editing:

1. Check `.codex/STATUS.md` for active claims.
2. Claim your investigation scope if you need to modify code or fixtures.

Do not:

- leave ambiguous reproduction state
- hand work back without exact verification steps and current risk notes
